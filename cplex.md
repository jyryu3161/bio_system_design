# IBM CPLEX solver 설치 

1. IBM CPLEX solver 다운로드

2. 설치파일 실행 권한 설정 
   chmod +x cplex_studio2211.linux-x86-64.bin

3. cplex 설치
/cplex_studio2211.linux_x86_64.bin # cplex 설치

cd /home/biosys/
